# OmopSketch: Characterise Tables of an OMOP Common Data Model Instance

Summarises key information in data mapped to the Observational Medical
Outcomes Partnership (OMOP) common data model. Assess suitability to
perform specific epidemiological studies and explore the different
domains to obtain feasibility counts and trends.
